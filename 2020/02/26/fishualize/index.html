A wonderful R package named fishualize

Feb 26, 2020 1 min read R

I recently discovered the R package fishualize by Nina Schiettekatte, Simon Brandl and Jordan Casey. This package is amazing ! The idea is that you use a set of colors taken from the picture of a fish species. Note that apparently, authors concentrate on teleost fishes although some chondrichthyans are present.

You can see the available color scales :

library(fishualize)
vignette("overview_colors")

…But the best is that you can also contribute your own favourite color scales!

vignette("contribute")

Using this simple procedure I fishualized one of my favorite fish, namely Serranus scriba (fam. Serranidae) which is a nice benthic fish very common in Mediterranean Sea and present in the western Atlantic and black sea as well. In Corsica, we call it “barchetta”. Its English name is “painted comber”.

From the picture below, I extracted the following colors :

#65071c

#b90b0a

#ef0d00

#72ccfe

#3d63a0

Serranus scriba. Source: Wikipedia

I hope this set of colors will be included in the next fishualize version.

Reference. Nina Schiettekatte, Simon Brandl and Jordan Casey (2019). fishualize: Color Palettes Based on Fish Species. R package version 0.1.0. https://CRAN.R-project.org/package=fishualize
